
# Installation instructions
---------------------------

Note: This extension requires that you have an account at areyouahuman.com.

1. Upload the folder tim_areyouahuman/ to your extensions/ directory.

2. Edit extensions/tim_areyouahuman/ayah_php_bundle_1.1.8/ayah_config.php

  define( 'AYAH_PUBLISHER_KEY', 'your_publisher_key_goes_here');
  define( 'AYAH_SCORING_KEY', 'your_scoring_key_goes_here');

3. Install your extension in the admin panel.

Note: This extension does not play well with Stop spam from bots 0.3.3

Enjoy!
